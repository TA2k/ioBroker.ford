FordConnect 2.0 OAuth

The adapter will display a URL in the log when started without authentication. Follow the instructions below to complete the OAuth login.

Login Instructions:
  1. IMPORTANT: Press F12 BEFORE opening the login URL to open Developer Tools
  2. Go to the Network tab in Developer Tools
  3. Copy the OAuth URL from the adapter log and paste it into your browser
  4. Log in with your Ford account
  5. After redirect, the browser will show "Cannot open page"
  6. Copy the complete URL from the browser address bar (starts with: fordapp://userauthorized/?code=)
  7. Paste it in the field below and save
  8. Restart the adapter

Paste the complete URL starting with "fordapp://userauthorized/?code=" here and save. The adapter will exchange this for an access token on next restart.
